# Detecao-de-Fraudes---Analise-de-Anomalias
Este projeto foi desenvolvido como parte dos estudos realizados no Bootcamp Bradesco da DIO (Digital Innovation One), com o objetivo de aplicar conceitos de Python, análise de dados e Machine Learning em um cenário próximo ao mercado financeiro e de segurança da informação.

A aplicação realiza a detecção de possíveis fraudes em transações financeiras por meio da identificação de comportamentos anômalos utilizando o algoritmo Isolation Forest. Além da análise estatística e comportamental, o sistema incorpora regras de negócio para avaliar fatores de risco como valor da transação, horário da operação, distância geográfica e quantidade de transações realizadas em um curto período.

Como resultado, a ferramenta gera um score de risco e classifica cada transação em níveis de criticidade, auxiliando processos de investigação e tomada de decisão. O projeto foi desenvolvido com foco educacional e tem como objetivo demonstrar conhecimentos em Python, Machine Learning, detecção de anomalias, análise de risco e desenvolvimento de soluções voltadas para prevenção a fraudes.
